# Роботы Чернобыля - интерактивный образовательный сайт

Одностраничный сайт для школьников и инженерных кружков о робототехнике при ликвидации аварии на Чернобыльской АЭС.

## Что внутри

- интерактивная обложка с аварийным роботом;
- хронология событий;
- карточки робототехнических комплексов: МОБОТ, СТР-1, Joker, Клин;
- образовательный блок про радиацию, связь, механику и отказоустойчивость;
- мини-симулятор риска;
- квиз на 5 вопросов;
- раздел с источниками.

## Как открыть

Если GitHub Pages включён для репозитория, сайт будет доступен по адресу:

`https://step3dlab.github.io/tehnopark_rgsu/`

Если страница пока не открывается, включите GitHub Pages:

1. Откройте репозиторий `STEP3DLab/tehnopark_rgsu`.
2. Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: `main`, folder: `/root`.
5. Save.

Также добавлен workflow `.github/workflows/pages.yml` для публикации через GitHub Actions, если в настройках Pages выбран режим GitHub Actions.

## Редактирование

Основной сайт находится в файле `index.html`.  
CSS и JavaScript встроены внутрь файла, чтобы сайт было удобно переносить и показывать без сборки.

## Назначение

Материал можно использовать как:

- учебный сайт;
- основу для комикса;
- демонстрационный стенд;
- интерактивный кейс для занятий по робототехнике, промышленному дизайну и инженерной безопасности.
