# Роботы Чернобыля - интерактивный образовательный сайт

Одностраничный сайт для школьников, инженерных кружков и музейно-образовательных показов о робототехнике при ликвидации аварии на Чернобыльской АЭС.

## Версия 3

Сайт доработан как полноценный интерактивный образовательный продукт, а не просто лендинг. Добавлены учебные сценарии, симуляторы и UX-механики для проведения занятия.

## Что внутри

- современная интерактивная обложка с аварийным роботом;
- адаптивный UX/UI под телефон, компьютер и проектор;
- липкая навигация с подсветкой активного раздела;
- индикатор прогресса чтения;
- тёмная и светлая тема;
- мобильное меню;
- быстрый поиск по разделам через кнопку поиска или Ctrl+K;
- режим урока, который оставляет только ключевые образовательные разделы;
- кнопка копирования ссылки;
- кнопка печати / сохранения в PDF;
- хронология событий с раскрывающимися пояснениями;
- карточки робототехнических комплексов: МОБОТ, СТР-1, Joker, Клин;
- выбор учебной роли: оператор, конструктор, дозиметрист, учитель;
- конструктор аварийного робота с параметрами защиты, проходимости, ремонта и связи;
- сценарная миссия оператора на 3 решения;
- образовательный блок про радиацию, связь, механику и отказоустойчивость;
- мини-симулятор риска с визуальной шкалой;
- квиз на 5 вопросов с прогрессом и подсветкой ответов;
- словарь терминов;
- раздел с источниками;
- кастомная страница 404;
- `robots.txt` и `sitemap.xml` для аккуратной публикации.

## Как открыть

Если GitHub Pages включён для репозитория, сайт будет доступен по адресу:

`https://step3dlab.github.io/tehnopark_rgsu/`

Если страница пока не открывается, включите GitHub Pages:

1. Откройте репозиторий `STEP3DLab/tehnopark_rgsu`.
2. Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: `main`, folder: `/root`.
5. Save.

Также добавлен workflow `.github/workflows/pages.yml` для публикации через GitHub Actions, если в настройках Pages выбран режим GitHub Actions.

## Редактирование

Основной сайт находится в файле `index.html`.

CSS и JavaScript встроены внутрь файла, чтобы сайт было удобно переносить, показывать и редактировать без сборки.

В коде оставлены комментарии к основным частям: стили, интерактивность, карточки роботов, симуляторы, миссия и квиз.

## Назначение

Материал можно использовать как:

- учебный сайт;
- основу для комикса;
- демонстрационный стенд;
- интерактивный кейс для занятий по робототехнике, промышленному дизайну и инженерной безопасности;
- прототип музейной или выставочной страницы про СКТБ ПР и аварийную робототехнику.
